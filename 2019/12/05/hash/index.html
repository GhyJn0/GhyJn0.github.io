哈希表

一次存取便能得到所查记录，那就必须在记录的存储位置和它的关键字之间建立一个确定的对应关系 f，使每个关键字和结构中一个唯一的存储位置相对应。

(关键字就是所要存储的数据，存储位置相当于数组的索引)

哈希表的特性

Hash 表是使用 O(1) 时间进行数据的插入删除和查找，但是 hash 表不保证表中数据的有序性，这样在 hash 表中查找最大数据或者最小数据的时间是 O(N)
注意
哈希表可以以极快的速度来查找、添加或删除元素（只需要数次的比较操作。）它比红黑树、二叉搜索树都要快得多，记录查找的频率。（huffeman树）但是哈希表没有排序功能，类似的，如寻找最大值、最小值、中值这些行为都不能在哈希表中实现。

hash 链表法存储示意图

hash链表法代码

public class Hash {
    //创建hashmap数组大小
    private static int size=4;
    public static void main(String [] args) {
        //定义一个hasH数组
        hashlist [] hashmap =new hashlist[size];
        for(int i=0;i<size;i++) {
            //初始化每一个hashmap 数组的元素
            hashmap[i]=new hashlist();
        }

        Node obj= new Node(1,"王");
        Node obj1= new Node(2,"li");
        Node obj2= new Node(3,"tian");
        Node obj3= new Node(5,"guang");
        //添加数据
        Add(obj,hashmap);
        Add(obj1,hashmap);
        Add(obj2,hashmap);
        Add(obj3,hashmap);
        //遍历hashmap
        List(hashmap);
    }
    //添加入指定的hashmap 中
    public static void Add(Node emp,hashlist[] hashmap) {
        int id=emp.id;
        int i=id%size;
        hashmap[i].add(emp);
    }
    //遍历hashmap中的所有链表
    public static void List(hashlist[] hashmap) {
        for(int i=0;i<size;i++) {
            System.out.print("数组"+i+"数据是:");
            hashmap[i].list();
            System.out.println();
        }
    }
}




class hashlist{
    //初始化链表头
    Node head=new Node(0," ");
    //添加数据
    public void add(Node emp) {

        if(head.next==null) {
            head.next=emp;
            return ;
        }
        Node temp=head;
        while( true) {
            if(temp.next!=null) {
                temp=temp.next;
            }
            else {
                break;
            }
        }
        temp.next=emp;


    }
    //遍历数据
    public void list() {
        Node temp=head.next;
        while(temp!=null) {
            System.out.print("数据:"+temp.id);
            temp=temp.next;

        }
    }
}

//节点信息
class Node{
    int id;
    String name;
    Node next;

    //构造器
    Node(int id,String name){
        this.id=id;
        this.name=name;
    }

}
