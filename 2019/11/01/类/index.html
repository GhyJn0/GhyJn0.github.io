学Java好‘类’

君の名は · 2019-11-1 · 次阅读

类 和对象

  • 类 ：是一组相关属性和行为的集合。可以看成是一类事物的模板，使用事物的属性特征和行为特征来描述该 类事物
  • 属性： 事物的状态信息 （变量）
  • 行为： 事物能做的事情（函数或者方法）

举例：小猫
属性：名字，体重，年龄。行为：走，跑，叫

  • 对象：不是那个男女朋友对象，是类的一个实例，并具备类的属性和行为

一类事物的一个实例：一只小猫
举例：一只小黄猫
属性：tom,10Kg,20 years.行为：溜墙跟走，蹦跶的跑，喵喵叫。

类的定义

  • 成员变量：对应事物的属性

  • 成员方法：对应事物的行为

  • 格式：
    public class ClassName{
    // 成员变量
    // 成员方法

    }

  • 定义类： 就是定义类的成员，包括成员变量和成员函数（方法）

  • 成员变量：和以前定义变量几乎是一样的。只不过位置发生了改变。在类中，函数（方法）外。

  • 成员方法：和以前定义方法几乎是一样。只不过把static去掉。

public class Student {
    //成员变量
    String name;
    int age;
    //成员方法
    public void Xuexi(){
        System.out.println("学习");
    }
    public void Game(){
        System.out.println("打游戏");
    }
}

对象

对象使用

  • 创建对象： 类名 对象名= new 类名（）；
  • 访问类中的成员
    对象名.成员变量；
    对象名.成员函数（）；

public class Student {
    //成员变量
    String name;
    int age;
    //成员方法
    public void Xuexi(){
        System.out.println("学习");
    }
    public void Game(){
        System.out.println("打游戏");
    }
}  
 public static void main (String[] arg)
    {
        Student test= new Student();
        //赋值成员变量
        test.name="王首都";
        test.age=20;
        //调用成员函数（方法）
        test.Game();
        test.Xuexi();
    }

默认成员变量

成员变量和局部变量

  • 在类中的位置不同
    成员变量：类中，方法外 局部变量：方法中或者方法声明上(形式参数)
  • 作用范围不一样
    成员变量：类中 局部变量：方法中
  • 初始化值的不同
    成员变量：有默认值 局部变量：没有默认值。必须先定义，赋值，最后使用
  • 在内存中的位置不同
    成员变量：堆内存 局部变量：栈内存
  • 生命周期不同 了解
    成员变量：随着对象的创建而存在，随着对象的消失而消失
    局部变量：随着方法的调用而存在，随着方法的调用完毕而消失

封装

  • 步骤：
    1. 使用 private 关键字来修饰成员变量。
    1. 对需要访问的成员变量，提供对应的一对 getXxx 方法 、 setXxx 方法。

private

  • 含义
    1.private 代表最小权限
    2.可以修饰成员变量和成员函数
    3.被private 修饰后的成员变量和成员函数，旨在类中才能访问。

使用格式

  • 格式
    private 数据类型 变量名；
    public class Student{
    private String name;
    private int age;
    }

  • 提供getXXX方法/setXXX方法,访问成员变量

设置类
public class Teacher {
    private int age;
    private String name;
    public void setName(String n)
    {
        name=n;
    }
    public void setAge(int a)
    {
        age=a;
    }
    public int getAge(){
        return age;
    }
    public String getName(){
        return name;
    }
}

初始化对象
public static void main(String [] args){
        Teacher Li= new Teacher();
        Li.setAge(10);
        Li.setName("李老师");
        System.out.println("年龄： "+Li.getAge());
        System.out.println("姓名： "+Li.getName());
    }

this

  • setXXX和getXXX 遗留的问题
    形参变量名与成员变量名重名: 导致成员变量名被隐藏，方法中的变量名，无 法访问到成员变量,赋值失败
public class Teacher {
    private int age;
    private String name;
    public void setName(String name)
    {
        name=name;
    }
    public void setAge(int age)
    {
        age=age;
    }
    }

/*年龄： 0
姓名： null*/
  • 含义
    this代表所在类的当前对象的引用（地址值），即对象自己的引用
    （方法被哪个对象调用，方法中的this就代表那个对象。即谁在调用，this就代表谁）
//方法中只有一个变量名时，默认也是使用 this 修饰，可以省略不写。 

public class Teacher {
    private int age;
    private String name;
    public void setName(String name)
    {
        this.name=name;
    }
    public void setAge(int age)
    {
        this.age=age;
    }
    public int getAge(){
        return age;
    }
    public String getName(){
        return name;
    }
}

构造方法

  • 当一个对象被创建时候，构造方法用来初始化该对象，给对象的成员变量赋初始值
  • 无论你与否自定义构造方法，所有的类都有构造方法，因为Java自动提供了一个无参数构造方法， 一旦自己定义了构造方法，Java自动提供的默认无参数构造方法就会失效。

格式

  • 修饰符 构造方法名（参数列表）{
    //方法体
    }
    注意： 方法名与它所在的类名相同。它没有返回值，所 以不需要返回值类型，甚至不需要void。

public class Teacher {
    private int age;
    private String name;

    //没有构造方法
    public Teacher(){}
    //有构造方法
    public Teacher(String name,int age){
        this.name=name;
        this.age=age;
    }
   }
  • 注意事项：
    1. 如果你不提供构造方法，系统会给出无参数构造方法。
    2. 如果你提供了构造方法，系统将不再提供无参数构造方法。
    3. 构造方法是可以重载的，既可以定义参数，也可以不定义参数

标准代码格式

public class ClassName{
    成员变量
    构造方法
    无参构造方法
    有参构造方法
    成员方法
    getXXX()
    setXXX()
}


public class Teacher {
    private int age;
    private String name;

    //没有构造方法
    public Teacher(){}
    //有构造方法
    public Teacher(String name,int age){
        this.name=name;
        this.age=age;
    }
    public void setName(String name)
    {
        this.name=name;
    }
    public void setAge(int age)
    {
        this.age=age;
    }
    public int getAge(){
        return age;
    }
    public String getName(){
        return name;
    }
}


//测试

public static void main(String [] args){
        //无构造方法
        Teacher Li=new Teacher();
        Li.setAge(10);
        Li.setName("李老师");
        System.out.println("年龄： "+Li.getAge());
        System.out.println("姓名： "+Li.getName());
        //有构造方法
        Teacher Wang= new Teacher("王老师",18);
        System.out.println("年龄： "+Wang.getAge());
        System.out.println("姓名： "+Wang.getName());


    }
