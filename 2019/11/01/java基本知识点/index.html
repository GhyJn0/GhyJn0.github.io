java基本知识点

君の名は · 2019-11-1 · 次阅读

命名

规则

  • 标识符可以包含 英文字母26个(区分大小写) 、 0-9数字 、 $（美元符号） 和 _（下划线） 。
  • 标识符不能以数字开头。
  • 标识符不能是关键字。

规范

  • 类名规范：首字母大写，后面每个单词首字母大写（大驼峰式）。
  • 方法名规范： 首字母小写，后面每个单词首字母大写（小驼峰式）。
  • 变量名规范：全部小写

基本数据类型

  • 基本数据类型： 整数，浮点数，字符，布尔。
  • 引用数据类型： 类，数组，接口。

练习

public class Variable { public static void main(String[] args){              
    //定义字节型变量         
    byte b = 100;         
    System.out.println(b);         
    //定义短整型变量         
    short s = 1000;         
    System.out.println(s);         
    //定义整型变量         
    int i = 123456;         
    System.out.println(i);         
    //定义长整型变量         
    long l = 12345678900L;         
    System.out.println(l);         
    //定义单精度浮点型变量         
    float f = 5.5F;         
    System.out.println(f);         
    //定义双精度浮点型变量        
     double d = 8.5;         
    System.out.println(d);         
    //定义布尔型变量         
    boolean bool = false;         
    System.out.println(bool);         
    //定义字符型变量         
    char c = 'A';         
    System.out.println(c); 
    }      
}

注意

  • long 类型：数据后面加L
  • float 类型： 数据后面加F

数据类型转换

1.自动转换

  • 定义：取值范围小的类型自动提升为取值范围大的类型
  • 规则：byte,short,char—> int—>long—>float—->double
public class ATS {
    public static void main( String [] args){
        int a=1;
        byte b=3;
        //byte c =b+a; 报错
        //int 类型和byte类型运算，结果是int 类型
        int c=a+b;
        System.out.println(c);
    }
}

2.强制类型转换

  • 格式：数据类型 变量名 = （数据类型）被转数据值；int a= (int)1.5

   public static void main (String [] args){
        /*short 两个字节，1是整形int 4个字节，
        出现编译失败        
        s和1做运算的时候，1是int类型，s会被提升为int类型        
        s+1后的结果是int类型，将结果在赋值会short类型时发生错误        、
        short内存2个字节，int类型4个字节        
        必须将int强制转成short才能完成赋值 
         */
        short s=1;
      s=s+1;//编译失败
        s=(short)(s+1);//编译成功
        System.out.println(s);

    }
}

注意

  • 浮点转成整数，直接取消小数点，可能造成数据损失精度。
  • int 强制转成 short 砍掉2个字节，可能造成数据丢失

  public static void main (String [] args){
        double x=1.1234343;
        //精度下降
        int b=(int)x;
        System.out.println(b);

        short a=32767;
        // 运算后，强制转换，砍掉2个字节后会出现不确定的结果
        short c=(short)(a+10);
        System.out.println(c);

    }

  • 格式：数据类型 变量名 = 布尔类型表达式？结果1：结果2

方法重载

  • 定义：指在同一个类中，允许存在一个以上的同名方法，只要它们的参数列表不同即可，与修饰符和返 回值类型无关。
  • 参数列表：个数不同，数据类型不同，顺序不同。 
public static void main(String [] args){
        int a=1,b=1;
        short c=1,d=1;
        byte e=1,f=1;
        long g=1L,h=1L;
        Cho(a,b);
        Cho(c,d);
        Cho(e,f);
        Cho(g,h);
    }
    public static void Cho(int a,int b){
        System.out.println("int :"+(a+b));
    }
    public static void Cho(short a,short  b){
        System.out.println("short :"+(a+b));
    }
    public static void Cho(byte a,byte b){
        System.out.println("byte :"+(a+b));
    }
    public static void Cho(long a,long b){
        System.out.println("long :"+(a+b));
    }

数组

定义

  • 方式一
    数组类型 [] 数组名字= new 数据类型[长度]
    int [ ] arr = new int[3]
  • 方式二
    数组类型 [] 数组名字= new 数据类型[ ]{元素1，元素2，…}
    int [ ] arr = new int[ ]{ 1,2,3,4,5}
  • 方式三
    数据类型[] 数组名 = {元素1,元素2,元素3…};
    int [] arr={1,2,3,4,5}
