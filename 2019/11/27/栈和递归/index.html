栈

栈的概念和基本操作

  • 概念： 
          栈顶：允许插入和删除的这一端称为栈顶。
  栈底：另外固定的这一端称为栈底。
  空栈：不含任何元素的栈称为空栈。
  • 特点：先进后出

栈代码

class ArrayStack{
    public int maxSize;//栈的大小
    private int [] stack;//栈
    public int top=-1;
    //构造器
    public ArrayStack(int maxSize) {
        this.maxSize = maxSize;
        stack= new int[ this.maxSize];
    }
    //判断栈是否为空
    public Boolean empty(){
        return top==-1;
    }
    //判断栈是否满
    public Boolean full(){
        return top==maxSize-1;
    }
    //把数据压入栈中
    public void push(int num){
        if(full()){
            System.out.println("栈满了");
            return;
        }
        top=top+1;
        stack[top]=num;
    }
    //取出数据
    public int pop(){
        if(empty()){
            throw new RuntimeException("栈空了");
        }
        int data=stack[top];
        top--;
        return data;
    }
    //遍历栈
    public void list(){
        while(top>=0){
            System.out.println(stack[top]);
            top--;
        }
    }

}

计算器

  • 步骤：
    • 1.声明一个数字的栈，一个符号的栈
    • 2.遍历字符串，数字添加到数字栈中，符号添加符号栈
    • 3.如果符号栈为空直接添加进去，如果符号栈已经有符号，需要比较新的符号和栈顶的符号的优先级
    • 4 * / 》 + -
    • 5.如果新的符号的优先级大于栈顶的优先级，把新的符号添加到符号栈顶
    • 否则符号栈pop栈顶，数字栈pop两个数据运算结果，数据栈加入结果，符号加入符号栈

    public int count(int num1,int num2,int a){
        int data=0;
        switch (a){
            case '+':
                data=num1+num2;

                break;

                data=num2-num1;

中缀表达式

  • 在日常应用中，算术表达式中运算符总是出现在两个操作数之间，这种形式称为中缀表达式，计算一个中缀表达式需要知道运算符的优先级和结合性。乘除是高优先级，加减是低优先级，优先级相同时他们都是左结合的，也就是从左计算到右。有括号就要计算括号内的表达式。
  • 例子：5（7-23）+8/2

后缀表达式（逆波兰）

  • 从左后缀表达式就是将运算符放在运算数之后。后缀表达式也称为逆波兰表达式。

  • 例子：9 3 1-3*+ 10 2/+

  • 后缀表达式计算-逆波兰计算：

    • 1.从左向右扫描，遇到数字压入栈，遇到运算符时，弹出栈顶的两个数
    • 2.计算结果压入栈
    • 3.重复过程直到遍历到最右端


    class Reckon{
        public String name;
        //构造器
        public Reckon(String name) {
            this.name = name;
        }
        // 返回一个后缀的列表
        public List<String> Visit(){
            List<String> list=new ArrayList<>();
            String [] Str=name.split(" ");
            for(String s:Str){
                list.add(s);
            }
            return list;
        }
        //计算
        public int Caculate(List<String> list){
            Stack<String > stack= new Stack<>();
            for(String l:list){
                //匹配多位数
                if (l.matches("\\d+")){
                    stack.push(l);
                }
                else{
                    //栈pop 出两个数
                    int a=Integer.parseInt(stack.pop());
                    int b=Integer.parseInt(stack.pop());
                    int res=0;
                    //注意此时不能用'' 符号要用""
                    if (l.equals("+")){
                        res=a+b;
                    }
                    else if(l.equals("-")){
                        res=b-a;
                    }
                    else if(l.equals("*")){
                        res=a*b;
                    }
                    else if(l.equals("/")){
                        res=b/a;
                    }
                    //System.out.printf("%d %s %d = %d\n",a,l,b,res);
                    //整形转为字符串
                    stack.push(""+res);
                }

            }
            //打印计算的结果
            System.out.println("结果为："+stack.peek());
            return Integer.parseInt(stack.pop());
        }

}

中缀表达转后缀表达式

  • 步骤：
    • 1.初始化2个栈： 运算符栈s1,和存储中间结果的栈s2
    • 2.从左到右扫面中追表达式
      1. 遇到操作数将其压入s2
    • 4.遇到运算符是，比较其与s1栈顶运算符的优先级
        1. 如果s1空，或栈顶运算符为左括号“（”,直接压入运算符s1栈，
        1. 否则，优先级比栈顶运算符的高，也将运算符压入运算符s1栈
        1. 否则 ，将s1栈顶的运算符弹出压入s2，在次转到（4，1） 与s1中的栈顶运算符比较
      1. 遇到括号时：
        1. 如果时“（”,直接压入运算符栈s1
        1. 如果是“）” ，则依次弹出s1栈订的运算符压入s2,知道遇到左括号为止，将这一对括号丢弃
    • 6.重复2-5,直到表达式最右边
    • 7.将s1中剩余的运算符一次弹出压入s2
    • 8 一次弹出s2中的怨妇输出，结果的逆序即为中缀转后追表达式

例子：1+((2+3)*4)-5

class MediumtoEnd{
    public String Formula;
    //构造器
    public MediumtoEnd(String formula) {
        this.Formula = formula;
    }

    public String getFormula() {
        return Formula;
    }

    // 转换成 后缀法
    public static String mediumSwitchend(String name ) {

        //初始化符号栈
        Stack<Character>s1= new Stack<Character>();
        //初始化数据存储栈
        Stack<String>s2= new Stack<String>();
        //初始化索引
        int i=0;
        // 拼接多位整数
        String c="";
        // 每一个字符
        char b;
        // 每一个字符紧挨着的一个字符
        char d=' ';

        while(i<=name.length()-1) {
            // 判断b是运算符
            b=name.charAt(i);
            if(Isoper(b)) {
                // s1是空或者b=='('直接加入
                if(s1.empty()|| b=='('){
                    s1.push(b);
                }
                else{
                    //如果是“）” ，则依次弹出s1栈订的运算符压入s2,知道遇到左括号为止
                    if (b==')'){
                        char w=s1.pop();
                        while(w!='('){
                            s2.push(w+" ");
                            w=s1.pop();
                        }
                    }
                    else{
                        char x= s1.peek();
                        //优先级比栈顶运算符的高，也将运算符压入运算符s1栈
                        if(Priopty(x)<Priopty(b)){
                            s1.push(b);
                        }
                        //
                        else{
                            //将s1栈顶的运算符弹出压入s2，在次转到（4，1） 与s1中的栈顶运算符比较
                            while(Priopty(x)>=Priopty(b)){
                                char  y=s1.pop();

                                s2.push(y+" ");
                                //如果此时操作符栈空停止，将b加入符号栈
                                if(s1.empty()){
                                    s1.push(b);
                                    break;
                                }
                                x=s1.peek();
                            }

                        }
                    }

                }
              //  System.out.println("符号是："+ b);
            }
            // 是操作数
            else {
                /*多位数*/
                //如果此时遍历到最后一位，不用判断下一位
                if(i==name.length()-1) {
                    c+=b;
                    s2.push(c);
                }
                //如果不是最后一位
                else {
                    c+=b;
                    d=name.charAt(i+1);
                    //下一位是运算符
                    if(Isoper(d)) {
                        //c 压入s2
                        c+=" ";
                        s2.push(c);
                        //System.out.println("s2 添加数据："+c);
                        // 把c清零
                        c="";
                    }
                }
            }
            i+=1;
        }
        // 遍历s1
        while(!s1.empty()){
            s2.push(" "+s1.pop()+" ");
        }
        String name1=list(s2);
        return name1;
    }
    //操作符的优先级 ：*/  >= -+
    public  static int Priopty (char c) {
        if(c=='*' || c== '/')
        {
            return 2 ;
        }
        else if ( c=='+' || c=='-') {
            return 1;
        }
        else {
            return 0;
        }
    }
    //判断是否是操作符
    public static  Boolean Isoper( char c) {
        return c=='*'|| c=='+'|| c=='-'|| c=='/'|| c=='(' || c==')';
    }
    // 返回转换后的后缀表达式
    public static String list( Stack<String > sk) {
        String name = "";
        while (!sk.empty()) {
            String q = sk.pop();
           // System.out.println("sk的数据：" + q);
            name += reverse(q);
        }
        return reverse(name);
    }
    //逆序字符串
    public static String reverse(String str){
        return new StringBuilder(str).reverse().toString();
    }
}

八皇后

  • 定义：所谓八皇后问题就是：将八位皇后放在一张8x8的棋盘上，使得每位皇后都无法吃掉别的皇后，（即任意两个皇后都不在同一条横线，竖线和斜线上），问一共有多少种摆法。

/*
* 1.第一个皇后先放第一行第一列
* 2.第二个皇后放第二行第一列，判断是否ok，如果不ok,继续放在第二列，第三列，意思把所有的列放完，周到一个合适
* 3. 继续第三个皇后，还是第一列，第二列。。。 直到第8个皇后，也能放在一个不冲突为值，算找到了一个正确姐
* 4.当得到一个正确解时，在栈会退到上一个时，就会开始回溯，即将第一个皇后，放到一列的所有正确解，全部找到。
* 5.回头机会第一个皇后放在第二列，后面继续循环1，2，3，4
* arr={0,4,7,5,2,6,1,3},arr下表 表示第几行，第几个皇后，arr[i]=val,val表示i+1个皇后，放在第val+
*简单说i表示行，val表示 列
* */
public class Queens8 {
    int  max=8;
    int [] arr=new int [max];
    int count=0;
    public static void main(String[] args) {
        Queens8 obj=new Queens8();
        obj.check(0);
        System.out.println("总共有"+obj.count+"种");
    }
    /**
     *放置第N个皇后
     * */
    public void check(int n){
        //如果n==8停止
        if (n==max){
            Print();
            count++;
            return;
        }
        //一次放入判断n列是否冲突
        for(int i=0;i<max;i++){
            //表示第n个在第i列
           arr[n]=i;
           //判断n是否冲突
           if(judge(n)){
               //不冲突继续判断第n+1
               check(n+1);
           }

        }
    }
    //放置第n个皇后，检测与之前的皇后是否冲突
    public  boolean judge(int n){
        for(int i=0;i<n;i++){
            //arr[i]表示列，所以arr[i]==arr[n]表示同一列，abs(n-i) ==abs(arr[n]-arr[i]);表示同一行
            if(arr[i]==arr[n]|| Math.abs(n-i)==Math.abs(arr[n]-arr[i]))
            {
                return false;
            }
        }
        return true;
    }
    //打印符合的
    public void  Print(){
        for(int i=0;i<arr.length;i++){
            System.out.print(arr[i]+" ");
        }
        System.out.println();
    }
}

        findQueen(0);
        System.out.println("八皇后问题共有："+map+"种可能");
    }

    public static void findQueen(int i){//寻找皇后节点
        if(i>7){//八皇后的解
            map++;
            print();//打印八皇后的解
            return;
        }

        for(int m=0;m<8;m++){//深度回溯,递归算法
            if(check(i,m)){//检查皇后摆放是否合适
                arry[i][m]=1;
                findQueen(i+1);
                arry[i][m]=0;//清零，以免回溯的时候出现脏数据
            }
        }
    }

    public static boolean check(int k,int j){//判断节点是否合适
        for(int i=0;i<8;i++){//检查行列冲突
            if(arry[i][j]==1){
                return false;
            }
        }
        for(int i=k-1,m=j-1; i>=0 && m>=0; i--,m--){//检查左对角线
            if(arry[i][m]==1){
                return false;
            }
        }
        for(int i=k-1,m=j+1; i>=0 && m<=7; i--,m++){//检查右对角线
            if(arry[i][m]==1){
                return false;
            }
        }
        return true;
    }

    public static void print(){//打印结果
        System.out.print("方案"+map+":"+"\n");
        for(int i=0;i<8;i++){
            for(int m=0;m<8;m++){
                if(arry[i][m]==1){
                    //System.out.print("皇后"+(i+1)+"在第"+i+"行，第"+m+"列\t");
                    System.out.print("o ");
                }
                else{
                    System.out.print("+ ");
                }
            }
            System.out.println();
        }
        System.out.println();
    }
