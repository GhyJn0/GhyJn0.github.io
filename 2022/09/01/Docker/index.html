Docker 安装

unbantu 安装docker

1，4，5

1.安装docker

# 1. 卸载docker
sudo apt-get remove docker docker-engine docker.io containerd runc
# 2. 更新包索引
sudo apt-get update
# 3. 安装依赖
sudo apt-get -y install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg-agent \
    software-properties-common
# 4. 添加 docker 官方 GPG 密钥
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
# 5. 添加安装 docker 软件源
# 使用阿里云的源比较快
sudo add-apt-repository "deb [arch=amd64] https://mirrors.aliyun.com/docker-ce/linux/ubuntu $(lsb_release -cs) stable"
# 6. 更新软件包索引
sudo apt-get -y update
# 7. 安装docker相关的 docker-ce 社区版 ee企业版
sudo apt-get -y install docker-ce docker-ce-cli containerd.io

2.特权指令运行容器

sudo docker run -tid --name tree --privileged=true ubuntu:16.04 /sbin/init
sudo docker exec -it tree /bin/bash

3. 下载安装vim

1.复制目前的源
mv /etc/apt/sources.list /etc/apt/sources.list.bak
2.修改源中科大

cat <<EOF >/etc/apt/sources.list
deb http://mirrors.ustc.edu.cn/debian stable main contrib non-free
deb http://mirrors.ustc.edu.cn/debian stable-updates main contrib non-free
EOF
3.更新源
apt-get update
4.下载vim
apt-get install vim
利用apt-get下载东西 到这为止
5.备份恢复
mv /etc/apt/sources.list.bak /etc/apt/sources.list

4.配置docker镜像加速

方法一
mkdir -p /etc/docker
tee /etc/docker/daemon.json <<-'EOF'
&#123;
  "registry-mirrors": ["https://mirror.ccs.tencentyun.com"]
&#125;
EOF
systemctl daemon-reload
systemctl restart docker

方法二
vim /etc/docker/daemon.json

&#123;
"registry-mirrors": [
  "https://mirror.ccs.tencentyun.com"
]
&#125;
sudo systemctl daemon-reload
sudo systemctl restart docker.service

service docker restart
docker info|grep Mirrors -A 1

🖊注意事项：所有操作在容器外操作，在root权限下进行。🖊

5.安装docker-compose

1.安装pip3
pip3 install pip -U -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
2.更新源
sudo apt-get update
3.安装docker-compose
pip3 install docker-compose
