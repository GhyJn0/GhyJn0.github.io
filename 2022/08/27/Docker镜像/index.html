镜像分层

Docker 镜像都是只读的，当容器启动时，一个新的可写层被加到镜像的顶部。

这一层就是我们通常说的容器层，容器层之下的都叫镜像层。

自定义镜像

docker commit [参数] 容器 [仓库[:标签]]

docker commit [参数] 容器 [仓库[:标签]]
参数

-a：作者信息。一般为 作者名字<邮箱>。
-c：将 Dockerfile 指令应用于创建的映像。
-m：注释信息。
-p：提交期间暂停容器（默认）
