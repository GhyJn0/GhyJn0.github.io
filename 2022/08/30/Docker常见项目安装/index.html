3.启动镜像 docker run –name nginx1 -p 3344:80 nginx

docker run --name nginx1 -p 3344:80 nginx
--name 取别名为nginx1
-p 主机端口 3344:容器端口 80 （nginx端口号为80） 实现主机端口到容器端口映射
